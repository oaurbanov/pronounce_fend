# pronounce_fend

Frontend of the pronounce app. Developed in react.

![app_img](./img/papp_fend.png)

## Getting Started

First, run the development server:

```bash
yarn install
yarn run dev
# or
npm build
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the page.
